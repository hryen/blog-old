## 简介

使用Spring Boot开发的自用博客系统

## 功能

* 代码高亮 文章置顶 文章回收站
* 友好的评论展示
* 附件批量上传 图片预览
* 可开启redis用来缓存首页及文章页等

## 技术栈

* Spring Boot
* MyBatis
* Redis
* Freemarker
* Element UI
* Vue.js
* axios.js
* Marked.js
* highlight.js
